# Attendance_Management_System
A Java-based Attendance Management System built using Java Swing for the GUI and MySQL for database management.
This application allows users to manage students, mark attendance, and view attendance records efficiently.

## Features

- Add new students

- Store student details in MySQL database

- Mark attendance (Present/Absent)

- View attendance records

- Simple and user-friendly GUI using Java Swing

- Persistent data storage with MySQL

 ## Concepts Used
  - Core Java
  - JavaSwing
  - MySQL
  - JDBC
